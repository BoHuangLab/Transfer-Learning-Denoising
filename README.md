- ```numpy==1.16.0```
- ```Pillow==5.4.1```
- ```scikit-image==0.14.2```
- ```scikit-learn==0.21.2```
- ```scipy==1.2.0```
- ```tensorflow-gpu==1.12.0```
- ```tifffile==2019.1.4```
- ```torch==1.1.0```
- ```torchvision==0.3.0```
- ```tornado==5.1.1```
